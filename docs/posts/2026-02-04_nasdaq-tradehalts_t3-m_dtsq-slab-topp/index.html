Halt code definitions (as listed by NASDAQ)

  • T3: “News and Resumption Times” — two times are displayed: (1) when market participants can enter quotations, and (2) when the security will be released for trading.
  • M: “Volatility Trading Pause” — trading has been paused in an Exchange-Listed issue (Market Category Code = C).
